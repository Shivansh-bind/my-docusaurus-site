## ðŸ”¸ The while Loop

Repeats while the condition is true.

### Syntax
```c
initialization;
while (condition) {
    // repeat code
    update;
}
```

### Example: Print Numbers
```c
int i = 1;
while (i <= 5) {
    printf("Number: %d\n", i);
    i++;
}
```

### Example: Sum of Natural Numbers
```c
int n, sum = 0, i = 1;
scanf("%d", &n);
while (i <= n) {
    sum += i;
    i++;
}
printf("Sum = %d\n", sum);
```

## ðŸ”¹ The do-while Loop

Executes **once before checking** the condition.

### Syntax
```c
do {
    // code
    update;
} while (condition);
```

### Example
```c
int y = 10;
do {
    printf("This will print at least once!\n");
} while (y < 5);
```

## ðŸ”¸ The for Loop

Best when the number of iterations is known.

### Syntax
```c
for (init; condition; update) {
    // repeat code
}
```

### Example: Multiplication Table
```c
int num;
scanf("%d", &num);
for (int i = 1; i <= 10; i++) {
    printf("%d Ã— %d = %d\n", num, i, num * i);
}
```

### Example: Factorial
```c
int n; unsigned long long fact = 1;
scanf("%d", &n);
for (int i = 1; i <= n; i++) {
    fact *= i;
}
printf("%d! = %llu\n", n, fact);
```

## ðŸŽ® Loop Control Statements

### break
Exits loop immediately.

```c
for (int i = 1; i <= 10; i++) {
    if (i == 5) break;  
    printf("%d ", i);
}
```

### continue
Skips current iteration.

```c
for (int i = 1; i <= 10; i++) {
    if (i % 3 == 0) continue;  
    printf("%d ", i);
}
```
